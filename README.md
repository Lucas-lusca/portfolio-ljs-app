# 📌 Portfólio – Desenvolvido com Angular 21

Bem-vindo ao repositório do meu novo portfólio!
Este projeto foi desenvolvido com o objetivo de apresentar minhas habilidades, experiências e alguns dos trabalhos que venho construindo ao longo da minha jornada como desenvolvedor.

## 🛠️ Tecnologias Utilizadas

- **Angular 21** — Framework principal utilizado na construção da aplicação  
- **TypeScript**
- **HTML5 & SCSS**
- **Docker** — Estudo e utilização de containers para organizar ambiente e etapas do desenvolvimento
- **GitHub Pages** — Plataforma utilizada para deploy da aplicação

---

## 📦 Objetivo do Projeto

Além de criar uma versão atualizada do meu portfólio, aproveitei este projeto para aprofundar conhecimentos em:

- Docker e gestão de containers  
- Fluxos de build e deploy  
- Organização de ambiente de desenvolvimento  
- Publicação de aplicações estáticas

O objetivo foi não apenas entregar um portfólio funcional, mas também **profissionalizar o processo de desenvolvimento e publicação**.

---

## 🚀 Deploy

A aplicação está hospedada no **GitHub Pages**, uma solução simples, gratuita e integrada ao ecossistema GitHub — ideal para aplicações estáticas como esta.
